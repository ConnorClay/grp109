Initial Project Details

This project aimed to create a website for a small cafe. The website was intended to have a professional and visually appealing design, showcasing the cafe's menu and potentially other information.

Key Features Planned:

  • Visually Appealing Design: The website would have a modern and professional design, including a background image and a fancy font for the title.
  • Interactive Element (JavaScript): A JavaScript-powered interactive element, such as a clickable tab in the corner, would reveal additional information about the cafe (e.g., backstory, values, beliefs). Note: The specific functionality of this element was still under consideration and might be adjusted based on complexity.
  • Nicely Arranged Menu: The cafe's menu would be displayed in a well-organized manner, with each menu item including:
    • A picture of the item
    • A description of the item
    • A link to a document (e.g., Word doc) listing the ingredients (if feasible)
  • Menu Layout: Menu items would be arranged in rows of two for a balanced and aesthetically pleasing presentation. We anticipated around six menu items initially, but this number might be adjusted.

Technical Considerations:

  • HTML would be used for the website's structure.
  • CSS would be used for styling and layout.
  • JavaScript would be used for the interactive element.
  • We would explore the feasibility of linking to external ingredient documents (Word docs or similar).

Project Goals and Outcomes

  • Goal: Create a visually appealing design, including a background image and fancy font. Outcome: Achieved. The website features a professional design, including a background image on the homepage, and consistent styling across pages. The use of fonts enhances the visual appeal.
  • Goal: Implement a JavaScript-powered interactive element (e.g., clickable tab). Outcome: Partially Achieved. While a clickable tab wasn't implemented, JavaScript is used for interactive elements like the order calculator on the menu page and the image carousel. The interactive element goal was modified to fit the project's scope and complexity.
  • Goal: Display the cafe's menu in a well-organized manner, with each item including a picture, description, and link to ingredients (if feasible). Outcome: Partially Achieved. The menu is well-organized with pictures and descriptions. However, linking to external ingredient documents was not feasible within the project's constraints.
  • Goal: Arrange menu items in rows of two. Outcome: Achieved. The menu items are arranged in a balanced and aesthetically pleasing grid layout, fulfilling the intent of the original goal.
  • Goal: Use HTML for structure, CSS for styling, and JavaScript for interactivity. Outcome: Achieved. HTML, CSS, and JavaScript were used effectively to build the website's structure, styling, and interactive elements.
